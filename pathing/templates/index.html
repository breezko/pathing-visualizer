Pathfinding Visualizer

How to Use

  1. Set Start Point: Click any cell to mark the starting position (green).
  2. Set End Point: Click another cell to mark the destination (red).
  3. Draw Walls: Click and drag to create obstacles (black).
  4. Select Algorithm: Choose A*, Dijkstra, or Breadth-First Search from the dropdown.
  5. Choose Speed: Select how fast the visualization plays (slow, normal, fast, or instant).
  6. Optionally: Generate Random Maze: Click the "Generate Maze" button to create a random obstacle layout. (Make sure you have set start and end first.)
  7. Solve: Click "Solve" to visualize the pathfinding algorithm.
  8. Clear: Reset the grid to start over.

The notification below the buttons will show how many nodes were explored and how long the computation took.
